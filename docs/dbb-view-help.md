# dbb view --help

## DESCRIPTION
 -----------
 Displays a metadata store object.
 Shorthand: 'vw'

### USAGE

 dbb view [argument] [options]
 Where [argument] is one of the following:

### ARGUMENTS

 g, group <groupName>

      Displays a build group.

 c, collection <groupName> <collectionName>

      Displays a collection.

 f, file <groupName> <collectionName> <fileName>

      Displays a logical file. 

 r, result <groupName> <resultLabel>

      Displays a build result.

 m, map <groupName> <fileName>

      Displays a build map.

### COMMAND OPTIONS

 -j,--json   Format the CLI output of view, find, list, or export commands, as
             well as any CLI errors encountered, as a JSON structure.

### CONFIGURATION OPTIONS

 NOTE! Configuration options can be omitted if CLI session has been started and
 active.

 -C,--db2-config <arg>      Location of Db2 connection configuration file. See
                            DB2's configuration file documentation for details
                            on its schema.
 -f,--password-file <arg>   Location of Db2 connection password stored in a
                            password file. IMPORTANT! The password file
                            contains an encrypted password and must be
                            generated by the DBB password utility.
 -h,--help                  Displays the CLI commands.
 -L,--location <arg>        File System Implementation location. Default is
                            '$HOME'.
 -p,--password <arg>        Db2 connection password. IMPORTANT! Password must
                            be encrypted using the DBB password utility.
 -t,--type <arg>            Metadata Store type. Values: file, db2. Default is
                            file.
 -u,--user <arg>            Db2 connection user ID.
 -U,--url <arg>             Db2 server location. Local or remote URL. Optional
                            if URL is provided in --db2_config file.

### LEGACY COLLECTIONS

 Legacy collections created in DBB 2.0.x releases are stored in a default DBB
 build group. You can use the deprecated commands to interact with legacy
 collections.
